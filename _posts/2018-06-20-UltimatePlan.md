## Windows 10 version 1803 或更高版本系统

快捷键 Win+ X → Windows PowerShell （管理员），然后输入下面的命令：

```
powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
```

打开「控制面板」→「系统和安全」→「电源选项」→「显示附加计划」就可以看到**「卓越性能」**
